Contact Chefle

Chefle is maintained as a small independent web game. The fastest public support path is the project issue tracker.

Support And Corrections

Use the GitHub issue tracker for bug reports, dish data corrections, broken image reports, accessibility feedback, policy questions, or requests to review content.

Open a Chefle support issue

Privacy And Advertising Questions

For privacy, cookie, consent, or advertising concerns, include the page URL, browser, approximate date, and a short description of what you saw. Do not post sensitive personal information in public reports.

Private Requests

If a request involves sensitive personal information, do not include those details in a public issue. Open a brief support issue asking for a private follow-up channel, or use the private contact method listed here once the site owner configures one.

Content Review Requests

If a food image, dish name, or description seems inaccurate or should be reviewed, include the dish title and the reason for the request. Chefle uses simplified gameplay data and can correct clear mistakes.
