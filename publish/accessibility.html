Chefle Accessibility

Chefle is intended to be usable with keyboard navigation, visible focus states, responsive layouts, reduced-motion preferences, and screen-reader-friendly status text where practical.

Current Support

  • Form controls and buttons can be reached by keyboard.
  • Dialogs use modal focus handling and can be closed with Escape.
  • Status updates use live regions where practical.
  • The app respects reduced-motion preferences.

Known Limitations

Chefle is a visual comparison game, so some gameplay depends on reading grid values, colors, and symbols. The text labels inside tiles are included to reduce color-only dependence.
